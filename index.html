php-inserterr Documentation

What is php-inserterr?

php-inserterr is a flexible php script for inserting HTML Form data into SQL Tables

php-inserterr will automatically detect the HTML form fields that are POSTed to it and will insert the data into a given SQL Table according to the form field names. If you add a new form field to your form (and a column to your SQL table), php-inserterr will automatically insert the form field's data into the related table column.

Usage

Setting up your HTMl form

When setting up your form, remember to name your form fields according to the column names in your SQL table! Otherwise php-inserterr won't be able to detect which SQL column the form field belongs to and will throw an error.

Options

php-inserterr provides provides a bunch of options you can define like this:

input type="hidden" name="php-inserterr-[option]" value="[value]">

Optional

Option Description
table Name of the SQL table php-inserterr will use.
redirect URL the user will be redirected to after the insertion.
key-prefix Prefix in your form-field names. Will be removed before being used as the SQL Table column names
column-prefix Prefix in your SQL table column names. Will be appended to your form-field names.
